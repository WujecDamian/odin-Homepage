# Odin Homepage

A responsive homepage built as part of [The Odin Project](https://www.theodinproject.com) curriculum.

**Lesson:** [Advanced HTML and CSS - Homepage](https://www.theodinproject.com/lessons/node-path-advanced-html-and-css-homepage)

## Overview

This project focuses on advanced layout techniques using **CSS Grid**, **Flexbox**, and **media queries** to create a fully responsive homepage that looks great on desktop, tablet, and mobile devices.

## Features

- Fully responsive design (mobile-first approach)
- Modern layout using CSS Grid and Flexbox
- Clean and semantic HTML structure
- 3main breakpoints for mobile -> tablet/laptop -> desktop
- Webpack for a professional development environment

## Challenges & Learnings

One of the biggest challenges I faced was **managing media queries**. Changing styles for the tablet view often affected the mobile layout, requiring me to explicitly re-define previous styles in smaller breakpoints.
